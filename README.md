# MochiBot

MochiBot is a discord bot to help you send cute kaomojis (emoticons derived from special characters) in your Discord channel.
Currently, 4 emotions are supported: happy, sad, angry, and love!

## Commands

Happy

```happy
~happy
```

Sad

```sad
~sad
```

Angry

```angry
~angry
```

Love

```love
~love
```
